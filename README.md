## 스터디 진행 방식과 규칙
### 진행 방식
- 매주 라이브코딩 세션/코드리뷰 세션 진행
- `라이브코딩 세션`은 문제 선정 후 풀이 + 각자의 풀이 방법 공유
- `코드리뷰 세션`은 각자 푼 문제 개념+알고리즘+풀이방식 설명
